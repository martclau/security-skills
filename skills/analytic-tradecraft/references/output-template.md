# Default output template

## Main judgment
One paragraph with the answer up front.

## Confidence
High / medium / low, plus the main reason for that level.

## Key assumptions
- assumption
- assumption
- assumption

## Leading hypotheses or alternatives
1. hypothesis
2. hypothesis
3. hypothesis

## Evidence and reasoning
- most diagnostic evidence
- disconfirming evidence
- gaps or ambiguities

## Indicators / signposts
- indicator that would increase confidence
- indicator that would reduce confidence
- indicator that would favor an alternative

## What could change the judgment
Short paragraph on reversibility conditions.
