 Book synthesis

This skill synthesizes three complementary strands:

- **Psychology of Intelligence Analysis**: mental models are unavoidable; analysts should be self-conscious about reasoning; explicit assumptions, alternative viewpoints, and tools like ACH help offset bias.
- **A Tradecraft Primer**: a practical toolkit for assumptions checks, evidence-quality review, indicators, ACH, devil's advocacy, red teaming, brainstorming, outside-in thinking, and alternative futures.
- **Thinking and Writing**: analytical limits are tied to working memory and heuristics; externalizing reasoning improves cognition; writing should be reader-centered, concise, and shaped around the conceptual front end of the problem.

## Core synthesis

1. Make the reasoning visible.
2. Challenge assumptions and hypotheses early.
3. Evaluate evidence quality, not just quantity.
4. Prefer disconfirmation over confirmation.
5. Track signposts that would update the judgment.
6. Write for the reader's limited attention and working memory.

## Recommended method map

- Ambiguous causal question -> ACH-lite + assumptions check
- Warning problem -> indicators/signposts + alternative futures
- Entrenched consensus -> devil's advocacy or red team
- Sparse/conflicted sourcing -> quality of information check
- Broad exploratory question -> brainstorming + outside-in thinking
- Executive deliverable -> minimum viable mode
